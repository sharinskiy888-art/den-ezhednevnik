# «День» — ежедневник

Адаптивное PWA-приложение для телефона и Windows. Работает офлайн, устанавливается как обычное приложение и может синхронизировать задачи и фотографии через Supabase.

## Запуск

Из папки приложения запустите локальный HTTP-сервер:

```powershell
python -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Основные возможности

- представления «год», «месяц», «неделя» и «день»;
- автоматический перенос невыполненных задач на следующий день;
- поиск и приоритеты без лишнего деления задач на категории;
- красная обводка и знак «!» у дней недели, где есть важные задачи;
- строгий деловой дизайн и порядок периодов «день — неделя — месяц — год»;
- компактный iPhone-режим без прокрутки всей страницы: прокручивается только длинный список задач;
- мобильный интерфейс в логике iPhone: системная типографика, сегментированный переключатель, группированный список и нижняя панель;
- персональная шапка с приветствием, круглой фотографией, именем, датой и количеством оставшихся дел;
- дневной главный экран и отдельная кнопка диапазона дат для перехода к полному плану недели;
- меню «Дела и планы» со списком на сегодня и единым списком долгосрочных планов;
- один план автоматически показывается в своей неделе, месяце и году независимо от вкладки, где он был добавлен;
- пункты планов с необязательным сроком, свободным выбором даты на много лет вперёд и навигацией по периодам;
- повторение задач ежедневно, по будням, еженедельно или ежемесячно;
- подзадачи с индикатором прогресса;
- быстрый ввод фраз вроде «Позвонить завтра в 10:30»;
- явный выбор «Без времени — в течение дня» или точного времени;
- завершение задачи галочкой с автоматическим перечёркиванием;
- голосовое создание задач;
- понятная проверка разрешения микрофона и резервный ввод через диктовку экранной клавиатуры;
- фотоотчёт, время снимка и комментарий к доказательству;
- напоминания и компактный список сегодняшних дел;
- отмена случайной отметки о выполнении;
- жесты на телефоне: вправо — выполнить, влево — изменить;
- отдельные личные аккаунты: каждый email видит только собственные задачи;
- безопасное переключение пользователей на одном компьютере без смешивания локальных данных;
- синхронизация личного ежедневника между телефоном и ноутбуком через Supabase;
- автономная работа после первого открытия.

## Установка

- **Windows / Android:** откройте приложение в Edge или Chrome и выберите «Установить приложение».
- **iPhone / iPad:** откройте в Safari, нажмите «Поделиться» → «На экран Домой».

## Облачная синхронизация

Нажмите значок облака, зарегистрируйтесь по email, а затем войдите с теми же данными на втором устройстве. Для жены, брата или другого человека создаётся отдельный аккаунт с другой почтой. На общем компьютере используйте кнопку «Сменить пользователя». Схема защищённой таблицы находится в `supabase-schema.sql`: каждый аккаунт видит только свои задачи.

Системные напоминания нужно один раз разрешить. Браузерные уведомления гарантированно срабатывают, пока приложение запущено; для доставки после полного закрытия потребуется отдельный сервер push-уведомлений.
